rcnR Documentation

Random Contaminated Normal Deviates

Description

Generates data sets from a contaminated normal distribution as in Example 4.9.2

Usage

rcn(n = 0, eps = 1e-04, sigma_c = 1)

Arguments

n

Size of random sample to generate

eps

Proportion of contamination

sigma_c

Standard deviation of contaminated component

Details

With probability (1-eps) a deviates are drawn from a standard normal distribution. With probability eps deviates are drawn from a normal distribution with mean 0 and standard devation sigma_c.

Value

n x 1 numeric vector containing the random deviates.

References

Hogg, R., McKean, J., Craig, A. (2018) Introduction to Mathematical Statistics, 8th Ed. Boston: Pearson.

Examples

rcn(n = 12, eps = 0.20, sigma_c = 4)
rcn(n = 30, eps = 0.30, sigma_c = 6)
