chistableR Documentation

Chi-Square Distribution

Description

Presents selected quantiles of the chi-square distribution.

Usage

chistable()

Details

This table more specifically produces the chi-square distribution table for the 0.01, 0.025, 0.050, 0.100, 0.900, 0.950, 0.975, and 0.990 quantiles. The initial row indicates the selected quantiles and the inital column indicates 'r' degrees of freedom See page 708 for more details.

Value

chistable is the chi-square distribution table for the selected quantiles.

References

Hogg, R., McKean, J., Craig, A. (2018) Introduction to Mathematical Statistics, 8th Ed. Boston: Pearson.

Examples

chistable()
